# Ai-Comments

It uses the [Next.js](https://nextjs.org/) framework with [React](https://reactjs.org/)

Deploy: https://chanel-test.netlify.app/

Node: v18.15.0

## Installation

Clone this repo and npm install.

```bash
npm i
```

## Usage

### Development server

```bash
npm run dev
```

You can view the development server at `localhost:5173`.

### Production build

```bash
npm run build
```
